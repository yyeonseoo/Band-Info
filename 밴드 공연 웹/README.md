# 밴드 공연 관리 웹

밴드 공연, 밴드, 공연장을 체계적으로 관리하는 웹 애플리케이션입니다.

## 주요 기능

- 📋 **엑셀 업로드**: 관리자가 엑셀 파일로 게스트 정보를 업로드
- 🔐 **모바일 체크인**: 이름과 전화번호로 간편하게 로그인
- 🎫 **티켓 정보**: 공연 티켓 정보 확인
- 🎵 **셋리스트**: 공연 곡 목록 확인
- 🎸 **공연진 정보**: 공연진 목록 확인
- 🎉 **이벤트**: 공연 관련 이벤트 정보 확인

## 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **React Router** - 라우팅
- **xlsx** - 엑셀 파일 읽기

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:5173`으로 접속할 수 있습니다.

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```

## 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── Layout.tsx       # 레이아웃 컴포넌트
│   ├── ProtectedRoute.tsx  # 보호된 라우트
│   ├── Ticket.tsx       # 티켓 컴포넌트
│   ├── Setlist.tsx      # 셋리스트 컴포넌트
│   ├── Performers.tsx   # 공연진 컴포넌트
│   └── Events.tsx       # 이벤트 컴포넌트
├── contexts/            # Context API
│   ├── AuthContext.tsx  # 인증 컨텍스트
│   └── DataContext.tsx  # 데이터 컨텍스트
├── pages/               # 페이지 컴포넌트
│   ├── Home.tsx         # 홈 페이지
│   ├── Login.tsx        # 로그인 페이지
│   ├── Admin.tsx        # 관리자 페이지
│   ├── Dashboard.tsx    # 사용자 대시보드
│   ├── Performances.tsx
│   ├── Bands.tsx
│   └── Venues.tsx
├── App.tsx              # 메인 앱 컴포넌트
├── main.tsx             # 진입점
└── index.css            # 전역 스타일
```

## 사용 방법

### 1. 관리자 페이지에서 게스트 정보 업로드

1. `/admin` 페이지로 이동
2. 엑셀 파일 업로드 (이름, 전화번호 컬럼 필요)
3. 공연 정보 설정 (셋리스트, 공연진, 이벤트, 티켓 정보)

### 2. 게스트 체크인

1. `/login` 페이지로 이동
2. 이름과 전화번호 입력
3. 로그인 성공 시 대시보드로 이동

### 3. 대시보드에서 정보 확인

- 티켓 정보
- 셋리스트
- 공연진 정보
- 이벤트 정보

## 엑셀 파일 형식

엑셀 파일에는 다음 컬럼이 포함되어야 합니다:
- `이름` 또는 `name` 또는 `Name`
- `전화번호` 또는 `phone` 또는 `Phone`
